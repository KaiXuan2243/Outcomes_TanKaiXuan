Describe the main future and capabilities found in modern web browsers.

Usually we will see the interface layout on the modern web browsers no big different with the older browser, maybe it just add some new future or design. the first is address bar that let users to enter the url, and the browser will search and show the relevent content following the key words. Second is the speed of browser, it is a trend and important for every users because Java Script is more important now, so we want the browser more fast as it can to display the page and data. Next is tag manager, user can organize, find the bookmark and search the history through the simple of table data. Besides that, it can update automatic and ask the permission before install the installation package. The last is the security of modern browser is better than previous. And the compatibilities also decided the performance of web browser. The compatibilities are important to browser because it will cause the website can work or not and the experience of users, it will indirect effect the quantity of users.
